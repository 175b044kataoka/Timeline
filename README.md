# Timelineの使い方(※要検索)
## 準備
1. [Project]の[Create] → [Timeline]
1. Timelineを[Hierarchy]にDrag & Drop = D&D
1. メニューの[Window]からTimeline Windowを表示
1. [Add]で[Activation Track]と[Animation Track]を作成
1. 上記二つに動かしたいObjectをD&D  

***
## Activation
- Timelineにドラッグで処理を実装
- 右端と左端で処理の長さを調節可能
- 中心をクリック長押ししながらクリックで移動が可能
